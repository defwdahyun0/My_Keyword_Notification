# 2. 프로젝트 개요

## 1) 개발내용

- 앱 : 리액티브 네이티브
- 아주대 ( ajou.ac.kr, 과 홈페이지, 단과대 홈페이지), 다른 대학들 홈페이지, 에브리타임
- ajou.ac.kr - 학사관리, 장학, ..
- 과 홈페이지 - 카테고리 x

## 2) 시나리오

1 사용자가 로그인

2 url을 입력 (아주대 한정)

3 카테고리 입력

4 key word 추가 (선택)

4-1 [AND] 다음 키워드는 모두 동시에 포함

4-2 [OR] 다음 키워드 중 하나 이상 포함

4-3 [NOT] 다음 키워드는 모두 제외

5 푸시 알림 서비스 (key word가 포함된 단어)-휴대폰 화면에 알림
